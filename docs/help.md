## Démarrage Rapide

### Première Utilisation

1. **Créer un compte**
   - Cliquez sur "S'inscrire" en haut à droite
   - Remplissez vos informations
   - Vérifiez votre email

2. **Générer votre premier persona**
   - Accédez à la page "Générateur"
   - Remplissez le brief marketing
   - Cliquez sur "Générer le Persona"

3. **Exporter vos résultats**
   - Téléchargez en PDF ou CSV
   - Partagez avec votre équipe

### Interface Principale

- **Dashboard** : Vue d'ensemble de vos personas
- **Générateur** : Création de nouveaux personas
- **Bibliothèque** : Gestion de vos personas existants
- **Analytics** : Statistiques et insights

## FAQ - Questions Fréquentes

### Général

**Q : Qu'est-ce qu'un persona marketing ?**
R : Un persona est un profil fictif représentant votre client idéal, basé sur des données réelles. Il inclut des informations démographiques, comportementales et psychographiques.

**Q : Combien de personas puis-je générer ?**
R : Cela dépend de votre plan :
- Gratuit : 3 personas par mois
- Professional : 50 personas par mois
- Enterprise : Personas illimités

**Q : Les personas sont-ils personnalisables ?**
R : Oui, vous pouvez modifier tous les éléments générés et ajouter vos propres informations.

### Génération

**Q : Combien de temps faut-il pour générer un persona ?**
R : Généralement 30 secondes à 2 minutes, selon la complexité du brief.

**Q : Puis-je utiliser mes propres données ?**
R : Oui, vous pouvez importer des données clients existantes pour enrichir vos personas.

**Q : Les personas sont-ils basés sur des données réelles ?**
R : Oui, ils utilisent des données comportementales et culturelles de Qloo Taste AI et des analyses marketing actuelles.

### Export et Partage

**Q : Quels formats d'export sont disponibles ?**
R : PDF (présentation) et CSV (données brutes).

**Q : Puis-je partager mes personas ?**
R : Oui, vous pouvez partager via lien ou export direct.

**Q : Les personas sont-ils sauvegardés automatiquement ?**
R : Oui, tous vos personas sont sauvegardés dans votre bibliothèque.

### Facturation

**Q : Comment changer de plan ?**
R : Allez dans "Paramètres" > "Abonnement" et choisissez votre nouveau plan.

**Q : Puis-je annuler à tout moment ?**
R : Oui, vous pouvez annuler votre abonnement à tout moment.

**Q : Y a-t-il des frais cachés ?**
R : Non, le prix affiché est le prix final. Aucun frais caché.

### Technique

**Q : Quels navigateurs sont supportés ?**
R : Chrome, Firefox, Safari, Edge (versions récentes).

**Q : L'application fonctionne-t-elle sur mobile ?**
R : Oui, l'interface est responsive et optimisée mobile.

**Q : Mes données sont-elles sécurisées ?**
R : Oui, nous utilisons un chiffrement SSL et respectons le RGPD.

## Résolution de Problèmes

### Problèmes Courants

**Erreur : "Génération échouée"**
- Vérifiez votre connexion internet
- Assurez-vous que votre brief est complet
- Réessayez dans quelques minutes

**Erreur : "Limite atteinte"**
- Vérifiez votre plan d'abonnement
- Attendez le renouvellement mensuel
- Contactez le support pour une augmentation

**Problème : Export ne fonctionne pas**
- Vérifiez que le persona est bien généré
- Essayez un autre navigateur
- Videz le cache de votre navigateur

### Optimisation des Performances

**Lenteur de génération**
- Simplifiez votre brief
- Évitez les descriptions trop longues
- Utilisez un navigateur récent

**Problèmes d'affichage**
- Actualisez la page
- Videz le cache
- Désactivez les extensions

### Support Technique

**Avant de contacter le support :**
1. Vérifiez cette FAQ
2. Consultez les guides d'utilisation
3. Testez sur un autre navigateur
4. Vérifiez votre connexion internet

**Informations à préparer :**
- Description du problème
- Étapes pour reproduire
- Captures d'écran si possible
- Informations système
